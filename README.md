# A-forms | CSS library to make beautiful custom forms

by Mateusz Ożóg

## How to use

In your html file

```
<!-- Import CSS -->
<link rel="stylesheet" href="/css/aforms.css">
```

Create **a-forms** compatible form

```
<form class="a-form">
    {...}
</form>
```
